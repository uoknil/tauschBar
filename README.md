# tauschBar – Software Engineering Projekt

tauschBar ist eine Plattform zum lokalen Tauschen von Fähigkeiten, Werkzeugen und kleinen Dienstleistungen. 
Die Anwendung basiert auf einem kombinierten Angebot/Gesuch-System und bietet Matching, Chat und Moderationsfunktionen.

Dieses Repository enthält:
- die gesamte Projektplanung,
- das Backlog (User Stories),
- Sprintplanung,
- Dokumentation,
- sowie später den vollständigen Code.

- **Frontend:** React oder Vue (Evaluierung in Sprint 1)

- **Datenbank:** MongoDB  
- **Kommunikation:** REST API  
- **Deployment:** lokal, später optional Cloud
